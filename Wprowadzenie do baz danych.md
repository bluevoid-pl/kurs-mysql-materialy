#### SQL - relacyjne

- Oparte na tabelach
- Implementują relacje jako podstawową funkcjonalność
- Zwykle używają języka SQL do zarządzania danymi i tablicami
- Zwykle dane są przechowywane w pamięci stałej (dysk twardy)
- MySQL, PostgreSQL, SQLite

#### noSQL - nie relacyjne

- Oparte na słowinikach, grafach i innych
- Dane mogą być przechowywane w pamięci ulotnej (RAM)
- MongoDB, Redis


#### Funkcjonalności SQL

- Tworzenie, odczytywanie, edycja i usuwanie baz danych, tabel i widoków.
- wyszukiwanie i przetwarzanie danych zawartych w bazie danych


#### Typy danych w bazach danych

- Text
- Liczba
